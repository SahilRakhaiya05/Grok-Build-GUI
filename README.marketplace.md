# Grok Build GUI

Independent VS Code sidebar for the Grok Build CLI over the Agent Client Protocol.

**Not affiliated with, endorsed by, or maintained by xAI.**

## Requirements

- VS Code 1.94+
- Grok CLI installed and signed in

## Features

- Agent chat sidebar with plan, agent, and YOLO modes
- File context, session history, and edit approvals
- Model picker and reasoning effort controls

## Install CLI

Download the Grok CLI from the official xAI site, then run `grok /login`.

## Settings

Configure `grok.cliPath` if the CLI is not on your PATH.
